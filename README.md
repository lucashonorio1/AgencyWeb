## 💻 Sobre o projeto

<!-- EXPLICA O MOTIVO DO PROJETO -->
{AgencyWeb} é um projeto criado no curso de Desenvolvedor Front End.

O AgencyWeb é um projeto de landing page institucional desenvolvido como parte da minha formação no curso de Desenvolvedor Front-End. O objetivo principal foi aplicar na prática conceitos essenciais de estruturação e estilização web, simulando um cenário real de mercado para uma agência digital.

Tecnologias e Técnicas Utilizadas

HTML5: Estruturação do conteúdo de forma clara e organizada.

CSS3: Estilização, design moderno e foco na experiência do usuário.

Tags Semânticas: Uso correto de boas práticas de semântica para garantir um código limpo, mais acessível e otimizado para motores de busca (SEO).
